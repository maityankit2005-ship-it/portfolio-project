# Project Overview – Static Portfolio Webpage

## 1. Introduction
This project is a simple static portfolio webpage created using only HTML and CSS.  
Its purpose is to display my basic information, skills, projects, and contact details in a clean and organized layout.

## 2. Project Goal
The main goal of this portfolio is:
- To present myself in a professional way  
- To showcase sample projects  
- To build a responsive webpage that works on mobile and desktop  
- To learn Git, GitHub, and VS Code workflow  

## 3. Tools & Technologies Used
- **HTML5** – structure  
- **CSS3** – styling & layout  
- **Visual Studio Code** – development  
- **Git** – version control  
- **GitHub** – project hosting  

## 4. Development Process
1. Created project folder in VS Code  
2. Added `index.html` with header, about, projects, and contact sections  
3. Designed layout using CSS flexbox/grid  
4. Added images and project cards  
5. Tested responsiveness using mobile view  
6. Initialized Git repository and made meaningful commits  
7. Pushed the final project to GitHub  
8. Added documentation, screenshots, and README
